

📌 Descripción del Proyecto
Este es un sistema de gestión de trabajos para "FOTO VIDEO MOJICA", donde diferentes usuarios (pc1, pc2, ..., pc9) pueden registrar trabajos pendientes y asignarlos a diseñadores previamente registrados. Cada trabajo se almacena en la tabla datos y puede ser marcado como listo, pasando a la tabla listos.

Roles del sistema:
✅ Administrador: Puede ver, eliminar y modificar todos los trabajos.
✅ Usuarios (pc1 - pc9): Diseñadores que ven y confirman como listos los trabajos asignados a ellos.
✅ servisio: Son usuarios que ven y supervisan trabajos.

📂 Base de Datos: pendiente
1️⃣ Tabla: datos (Registra los trabajos pendientes)
Campo	Tipo	Descripción
id	INT (PK)	Identificador único del trabajo
descripcion	VARCHAR(255)	Detalles del trabajo
diseñador	VARCHAR(255)	Nombre del diseñador asignado
diseñador_id	INT (FK)	Relación con la tabla diseñadores
no_factura	VARCHAR(50)	Número de factura
cliente	VARCHAR(50)	Nombre del cliente
fecha	DATE	Fecha del registro
numero de telefono	VARCHAR(50)	Teléfono del cliente
📌 Importante: Esta tabla usa la clave foránea diseñador_id para vincular el trabajo con un diseñador específico.

2️⃣ Tabla: diseñadores (Lista de diseñadores disponibles)
Campo	Tipo	Descripción
id	INT (PK)	Identificador único del diseñador
nombre	VARCHAR(50)	Nombre del diseñador
usuario	VARCHAR(50)	Usuario vinculado (ej. pc1, pc2, etc.)
es_admin	TINYINT(1)	1 si es admin, 0 si es diseñador
📌 Importante: Los diseñadores son usuarios del sistema y pueden ser asignados a trabajos.

3️⃣ Tabla: listos (Guarda trabajos finalizados)
Campo	Tipo	Descripción
id	INT (PK)	Identificador único del trabajo listo
descripcion	VARCHAR(255)	Descripción del trabajo
diseñador	VARCHAR(100)	Nombre del diseñador
no_factura	VARCHAR(50)	Número de factura
cliente	VARCHAR(100)	Nombre del cliente
fecha	DATE	Fecha de finalización
numero de telefono	VARCHAR(20)	Teléfono del cliente
📌 Importante: Solo se almacenan trabajos que ya fueron completados.

🖥️ Páginas del Proyecto
1️⃣ login.php → Página de inicio de sesión para validar usuarios.
2️⃣ edit.php → Panel principal donde se registran nuevos trabajos.
3️⃣ trabajos_listos.php → Muestra los trabajos finalizados.
4️⃣ pcX.php (pc1, pc2, ..., pc9) → Panel donde cada usuario ve sus trabajos.
5️⃣ admin.php → Panel exclusivo para el administrador.
6️⃣ servicio.php → Interfaz para el equipo de servicio técnico.
7️⃣ conexion.php → Archivo para conectar a la base de datos.
8️⃣ styles.css → Archivo CSS para mejorar la apariencia.
